# DonorDeck - Emergency Blood Bank Management System
A comprehensive Emergency Blood Bank Management System developed as a university project for the Object-Oriented Programming course. This system integrates advanced programmatic structures to manage, track, and optimize blood donor and recipient records efficiently.

## Project Overview
- Backend Core: Developed in C++ using Object-Oriented Programming concepts to handle core data management and logical processing.
- Frontend Web Interface: An intelligent, user-friendly dashboard layout showcasing live system architecture, donor modules, and administrative metrics.

## System Features
- Donor Management: Comprehensive modules to add, view, search, update, and manually delete donor profiles.
- Recipient Management: Full tracking system to register recipients, manage search requests, and log history.
- Blood Inventory Control: Real-time stock visibility and blood request processing.
- Intelligence Module: Features blood compatibility analysis, transaction mapping, and a settings panel equipped with data auto-saving protocols alongside manual override data deletion.

## File Structure
- DonerDeck.cpp: The primary C++ source file containing all system logic, class structures, and database simulation.
- index.html: The core web layout file rendering the visual command center and menu navigation.
